# 🌟 Initial Release - Advanced Netherite (Bedrock Port)

Welcome to the brand new release of **Advanced Netherite** for Minecraft Bedrock Edition! This version marks a complete ground-up rework by **Lilium Studio**, bringing the beloved Java Edition mod by _Autovw_ to your Bedrock worlds with unparalleled quality and performance.

### ✨ Key Features

- **4 New Netherite Tiers:** Expand your endgame progression with Netherite-Iron, Netherite-Gold, Netherite-Emerald, and Netherite-Diamond.
- **Complete Sets:** Every tier includes a full armor set and a complete toolset (Sword, Spear, Pickaxe, Axe, Shovel, and Hoe).
- **Unique Properties:** Each material tier comes with its own distinct properties and advantages.
- **Vanilla-Friendly:** Designed to fit seamlessly into the vanilla Minecraft experience.

### 🛠️ Technical Details

- **No Experimental Toggles:** Just plug and play! You don't need to enable any experimental features in your world settings.
- **Maximum Compatibility:** This addon **DOES NOT** use `player.json`, making it highly compatible with almost any other addon.
- **Modern Scripting Framework:** Completely rebuilt using the latest Bedrock Scripting API for improved performance, event-driven logic, and stability.
- **Entity Behaviors:** Uses custom components on `phantom.json`, `piglin.json`, and `enderman.json` to maintain vanilla mechanics (e.g., Piglins won't attack you while wearing Netherite-Gold armor).

### 🎨 Addon Compatibility

Fully compatible right out of the box with:

- **Colorful Armor Bar** (by Dimzcraft)
- **Durability Armor Viewer** (by Dimzcraft)
