# Advanced Netherite – Bedrock Edition

![Banner](https://i.postimg.cc/4dzJ9yxX/Advanced-Netherite-Banner-V13.png)

**Advanced Netherite** is a vanilla-friendly, open-source mod originally created for Java Edition.  
This project is an **unofficial Bedrock Edition port** that brings the same extended Netherite progression to Minecraft Bedrock.

---

## 📜 Permissions & Credits

This port is an **authorized unofficial** version.  
The original Java mod creator, **Autovw**, has granted permission to create this Bedrock Edition adaptation.

**Permission proof:**  
https://github.com/Autovw/AdvancedNetherite/issues/78

**Original textures:** LizterZapZap (Java Edition)  
**Bedrock port:** Memorie Studio

---

## ✅ Features

![Features](https://i.postimg.cc/MTd6yX4W/Advanced-Netherite-Features-V11.png)

- 4 new Netherite tiers: Iron, Gold, Diamond, Emerald
- Full armor sets for each tier
- Full tool sets (Sword, Pickaxe, Axe, Shovel, Hoe)
- Vanilla-friendly progression

---

## 🛠️ Port Specifications (Bedrock)

- **No Experiments required**
- **Compatible version:** 26.0 (1.20.60+) and newer
- **High compatibility:** does **not** use `player.json`
- **Modified entities:** `phantom.json`, `piglin.json`, `endermen.json`  
  (e.g. Piglins ignore players wearing Gold-Netherite armor)

---

## ⚠️ Bedrock Limitations

Due to engine limitations, some Java features are not available:

- **Smithing Table:** upgrades are done via Crafting Table
- **Knockback Resistance:** not supported by the current API
- **Beacon bases:** custom blocks cannot be used
- **Translations:** item lore is not localized
